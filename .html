HTMLLinguagem de marcação de hipertexto

Tutoriais para iniciantes

HTML (Linguagem de Marcação de HiperTexto) é o bloco de contrução mais básico da web. Define o significado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

Tag Para que serve
TITLE Define o título que aparecerá na guia do navegador
HTML Define o escopo de um documento HTML
HEAD Define como vai ser o cabeçalho/título do documento
BODY Define o que será colocado dentro do documento(imagens, textos etc.)
a Define um Link que pode ser acessado pelo cliente
p Define um parágrafo do documento
!DOCTYPE HTML Define o tipo de documento para Html
Img Define uma imagem no documento
